## 🏗️ About This Repo

All projects here are built using **no-code or low-code** platforms and approaches.  
This repo serves as a **portfolio and archive** of such builds.

**Design philosophy:** Every project uses OOP-first backend design, local-first storage, and zero cloud subscription cost.
